## 欧奈尔基本面+技术分析

欧奈尔将强势股票在上涨初期表现出来的7种特征总结为CANSLIM,其中每一个字母代表一种特征。

| 字母  | 含义 |
| ------------- | ------------- |
| C  | 可加速增长的每股收益率 |
| A  | 靠前的年度收益增长率 |
| N  | 新公司，新产品，新管理层，股价新高 |
| S  | 股票需求大于供给 |
| L  | 股票在行业中的龙头地位 |
| I  | 机构认同度高 |
| M  | 市场走牛 |
